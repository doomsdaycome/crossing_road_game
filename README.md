### Release

You can download our game in release section and run it by using [Execution](#execution)

### Manual

```bash
git clone https://github.com/doomsdaycome/crossing_road_game.git
cd ./crossing_road_game/
mkdir builds/
cmake ../ && cmake --build ./
```

## Execution

```bash
./builds/crossing
```
